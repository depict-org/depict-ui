---
title: Installation
---

<Warning>
This page does not apply to installs made in 2025 or later of the Depict Shopify apps
</Warning>

The Depict JS UI package is available in the NPM registry. To use it, you must first install it:

<CodeGroup>

    ```shell NPM
    npm install @depict-ai/js-ui
    ```

    ```shell Yarn
    yarn add @depict-ai/js-ui
    ```
</CodeGroup>
