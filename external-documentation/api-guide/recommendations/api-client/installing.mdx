---
title: "Installation"
---

Start by installing a Depict UI package:

<CodeGroup>
    ```shell NPM (React)
    npm i @depict-ai/react-ui
    ```

    ```shell Yarn (React)
    yarn add @depict-ai/react-ui
    ```

    ```shell NPM (JS)
    npm i @depict-ai/js-ui
    ```

    ```shell Yarn (JS)
    yarn add @depict-ai/js-ui
    ```
</CodeGroup>
