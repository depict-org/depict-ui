---
title: "Types of product listing pages"
---


| Listing page | Description                                                                                              | Type in API Response | Can be requested with `externalId` | Created automatically |
|--------------|----------------------------------------------------------------------------------------------------------|----------------------|------------------------------------|-----------------------|
| Campaign     | Listing pages created inside of Depict by the merchant                                                   | campaign             | ❌                                  | ❌                     |
| Category     | Listing pages created inside of Depict by the merchant, or synced in from an external resource           | category             | ✅                                  | ❌                     |
| Smart Pick   | List pages automatically created by Depict using data driven rules such as "best sellers" or "trending". | smart_pick           | ❌                                  | ✅                     |
| Brand        | Pages automatically created by Depict for every brand                                                    | brand                | ❌                                  | ✅                     |
