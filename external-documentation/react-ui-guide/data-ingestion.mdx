Before you continue, make sure you have read and followed the [Data Ingestion](/data-ingestion-guide) guide.

Typically, there's a delay between the time of data ingestion and when the data becomes available in the API. If you want to verify the availability of the data, use the [preview browser](https://demo.depict.ai).
